# 微信小程序 IP 白名单限制问题

## 🚨 发现的问题

### 问题 1：IP 白名单数量限制
- **微信限制**：最多只能配置 **10 个 IP**
- **扣子平台**：官方提供 **25 个固定 IP**
- **实际情况**：还可能使用动态 IP

### 问题 2：动态 IP 问题
**已发现的不在官方列表中的 IP**：
```
115.191.1.197
101.126.128.57
101.126.95.49
115.190.192.22  ← 新发现
```

这些 IP 不在扣子官方提供的 25 个固定 IP 列表中。

---

## 💡 解决方案

### 方案一：多次尝试上传（推荐尝试）

**原理**：扣子平台可能随机选择出口 IP，多次尝试可能命中已配置的 IP

**操作**：
1. 保持当前 10 个 IP 配置
2. 多次执行上传命令
3. 观察是否能成功

**优点**：
- 无需修改配置
- 可能偶然成功

**缺点**：
- 成功率不确定
- 不稳定
