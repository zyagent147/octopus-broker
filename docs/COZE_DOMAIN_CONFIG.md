# 扣子平台配置服务器域名指南

## 🚨 当前情况

小程序已托管给第三方平台（扣子平台），无法在微信公众平台直接配置服务器域名。

---

## 📋 解决方案

### 方法 1：在扣子平台配置（推荐）

1. 登录 **扣子平台**：https://www.coze.cn
2. 进入你的项目：**房产经纪人小助手**
3. 找到 **微信小程序配置** 或 **服务器域名配置**
4. 添加以下域名到 `request 合法域名`：
   ```
   https://c95a3f35-a531-4fd7-b122-1faa2d7ff161.dev.coze.site
   ```

### 方法 2：联系扣子平台客服

如果找不到配置入口，联系扣子平台客服：

- 在扣子平台右下角点击 **客服** 图标
- 或发送邮件到：support@coze.cn
- 说明需要配置微信小程序服务器域名

### 方法 3：取消托管（不推荐）

如果需要完全控制小程序配置，可以在微信公众平台：
1. **开发管理** → **开发设置**
2. 找到 **第三方平台管理**
3. 取消扣子平台的托管授权
4. 然后手动配置服务器域名

⚠️ 注意：取消托管可能影响扣子平台的其他功能

## 📱 开发调试（临时方案）

在等待域名配置期间，可以使用微信开发者工具调试：

1. 打开微信开发者工具
2. 导入项目：选择 `dist-weapp` 目录
3. AppID：`wxd244b605ba704aab`
4. 点击右上角 **详情**
5. 勾选 **不校验合法域名、web-view（业务域名）、TLS版本以及HTTPS证书**

这样可以在开发环境中正常调试。
