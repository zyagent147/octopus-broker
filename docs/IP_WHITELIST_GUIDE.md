## ⚠️ 重要提示

### 1. IP 地址格式
- ✅ 正确：`101.126.128.57`
- ❌ 错误：`101.126.128.57:8080`（不要加端口）
- ❌ 错误：`http://101.126.128.57`（不要加协议）

### 2. 白名单作用
- 只允许列表中的 IP 上传代码
- 其他 IP 上传会被拒绝
- 影响命令行上传和自动化部署

### 3. 动态 IP 问题
如果扣子平台使用动态 IP：
- 需要添加整个 IP 段
- 或联系扣子平台获取固定 IP
- 或使用微信开发者工具上传（无需 IP 白名单）

### 4. 本地开发
本地开发时不受 IP 白名单限制：
- 微信开发者工具上传无需配置
- 只有命令行上传需要 IP 白名单

## 🆘 常见问题

### Q1: IP 白名单在哪里找到？
**A**: 
```
微信公众平台 → 开发管理 → 开发设置 → 小程序代码上传 → IP白名单
```

### Q2: 需要添加几个 IP？
**A**: 取决于扣子平台的服务器节点数量，建议联系扣子客服获取完整列表。

### Q3: IP 白名单影响什么？
**A**: 
- ✅ 影响：命令行上传、CI/CD 自动化部署
- ❌ 不影响：微信开发者工具上传

### Q4: 如何知道当前服务器 IP？
**A**: 
- 联系扣子平台客服
- 或在部署时查看错误日志

### Q5: 配置后多久生效？
**A**: 立即生效，无需等待。

### Q6: 可以添加 IP 段吗？
**A**: 不可以，只能添加单个 IP 地址。如果有多个 IP，需要逐行添加。
