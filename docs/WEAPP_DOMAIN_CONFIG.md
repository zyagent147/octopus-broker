# 微信小程序服务器域名配置指南

## 🚨 问题：白屏原因

微信小程序无法发起网络请求，因为**服务器域名未配置**。

---

## 📋 配置步骤

### 1. 登录微信公众平台

访问：https://mp.weixin.qq.com

使用小程序管理员账号登录

### 2. 进入开发设置

路径：**开发管理** → **开发设置** → **服务器域名**

### 3. 配置 request 合法域名

点击 **修改** 按钮，添加以下域名：

```
https://c95a3f35-a531-4fd7-b122-1faa2d7ff161.dev.coze.site
```

### 4. 保存配置

点击 **保存并提交** 按钮

---

## ⚠️ 注意事项

1. **域名格式**：必须以 `https://` 开头
2. **域名数量**：最多可配置 20 个 request 合法域名
3. **生效时间**：配置后立即生效，无需重新发布
4. **开发环境**：在微信开发者工具中可以勾选"不校验合法域名"进行测试

---

## 🧪 验证配置

配置完成后，在微信公众平台 **开发管理** → **开发设置** → **服务器域名** 中确认：

- ✅ request 合法域名中包含 `c95a3f35-a531-4fd7-b122-1faa2d7ff161.dev.coze.site`

---

## 📱 重新预览

配置完成后：

1. 在微信开发者工具中刷新预览
2. 或重新扫码预览
3. 点击登录按钮测试

---

## 🔍 其他可能问题

### 1. 后端服务未启动

确认后端服务正常运行：
```bash
curl https://c95a3f35-a531-4fd7-b122-1faa2d7ff161.dev.coze.site/api/health
```

应该返回：`{"status":"success","data":"..."}`

### 2. 微信登录配置

确认后端配置了微信小程序 AppID 和 Secret：
- `WX_APP_ID=wxd244b605ba704aab`
- `WX_APP_SECRET=ca31d883...`（已配置）
