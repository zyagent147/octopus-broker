# 扣子小程序部署指南

## 🚀 部署方式说明

扣子（Coze）平台**支持小程序一键部署**，但需要完成以下配置：

---

## ✅ 已支持的功能

### 1. H5 版本自动部署 ✅
- **开发环境**：`https://{project-id}.dev.coze.site`
- **生产环境**：`https://{project-id}.prod.coze.site`
- 部署时会自动执行 `pnpm build`，包含 H5 和后端

### 2. 后端服务自动部署 ✅
- 自动启动 NestJS 服务
- 支持热更新和日志查看

### 3. 小程序代码构建 ✅
- 自动构建小程序代码到 `dist-weapp` 目录
- 支持 Tailwind CSS 编译

## ⚠️ 小程序上传需要手动配置

微信小程序上传到微信平台**需要额外的密钥配置**：

### 第一步：获取上传密钥

1. 登录微信公众平台：https://mp.weixin.qq.com
2. 进入 **开发管理** → **开发设置**
3. 找到 **小程序代码上传** 部分
4. 点击 **生成** 生成上传密钥
5. 下载 `private.wxappid.key` 文件

### 第二步：配置 IP 白名单

在同样的页面，将服务器 IP 添加到白名单：
- 开发环境：需要添加 Coze 平台的出口 IP
- 生产环境：需要添加生产服务器 IP

### 第三步：配置环境变量

在项目根目录创建 `.env.local` 文件：

\`\`\`bash
# 微信小程序上传配置
TARO_APP_WEAPP_APPID=wxd244b605ba704aab

# 上传密钥路径（相对于项目根目录）
TARO_APP_WEAPP_PRIVATE_KEY_PATH=key/private.wxd244b605ba704aab.key
\`\`\`

### 第四步：放置密钥文件

将下载的密钥文件放到项目目录：
\`\`\`
/workspace/projects/key/private.wxd244b605ba704aab.key
\`\`\`

## 📋 部署流程对比

### 方式一：扣子平台一键部署（推荐）

**优点**：
- 自动构建代码
- 自动部署 H5 和后端
- 环境隔离（开发/生产）

**缺点**：
- 小程序上传需要配置密钥

**操作步骤**：
1. 配置上传密钥（见上文）
2. 点击平台"部署"按钮
3. 等待构建完成
4. 小程序代码会自动上传到微信平台

---

### 方式二：微信开发者工具上传（手动）

**优点**：
- 无需配置密钥
- 可视化预览和调试

**缺点**：
- 需要手动操作
- 无法自动化

**操作步骤**：
1. 打开微信开发者工具
2. 导入项目（选择 `dist-weapp` 目录）
3. 点击 **上传** 按钮
4. 填写版本号和备注
5. 点击确定完成上传

---

### 方式三：命令行上传（CI/CD）

**优点**：
- 可集成到 CI/CD 流程
- 自动化程度高

**缺点**：
- 需要配置密钥和 IP 白名单

**操作步骤**：
\`\`\`bash
# 构建
pnpm build:weapp

# 上传（需要配置密钥）
npx miniprogram-ci upload \\
  --pp ./dist-weapp \\
  --pkp ./key/private.wxd244b605ba704aab.key \\
  --appid wxd244b605ba704aab \\
  -r 1.0.0 \\
  --uv 1.0.0 \\
  --desc "首次发布"
\`\`\`

## 📝 快速开始（手动上传）

1. **构建小程序代码**：
\`\`\`bash
pnpm build:weapp
\`\`\`

2. **打开微信开发者工具**：
   - 导入项目
   - 选择 `dist-weapp` 目录
   - AppID：`wxd244b605ba704aab`

3. **上传代码**：
   - 点击右上角 **上传**
   - 填写版本号：`1.0.0`
   - 填写备注：`首次发布`
   - 点击确定

4. **提交审核**：
   - 登录微信公众平台
   - 版本管理 → 开发版本
   - 点击 **提交审核**

## 🆘 常见问题

### Q1: 扣子平台支持自动上传小程序吗？
**A**: 支持，但需要先配置上传密钥和 IP 白名单。配置后可实现一键部署。

### Q2: 密钥文件放在哪里？
**A**: 项目根目录的 `key/` 文件夹，文件名格式：`private.{appid}.key`

### Q3: IP 白名单怎么填？
**A**: 需要联系扣子平台获取服务器出口 IP，添加到微信公众平台白名单。

### Q4: 没有密钥能部署吗？
**A**: H5 版本可以自动部署，小程序需要手动上传或配置密钥。
