# 章鱼经纪人小程序 - 使用指南

## 🚀 如何运行这个项目

1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 打开工具，点击「导入项目」
3. 项目目录选择：`C:\Users\45207\Desktop\ZYagent\ZYagent-miniprogram`
4. AppID：`wxd244b605ba704aab`
5. 点击「确认」导入并预览

## 📂 目录结构

```
ZYagent-miniprogram/
├── app.js                    全局入口 + 演示数据
├── app.json                  页面路由 + TabBar 配置
├── app.wxss                  全局样式
├── pages/
│   ├── home/                 首页（统计卡片+今日提醒+快捷入口）
│   ├── client/               客户列表（搜索+标签筛选）
│   ├── client-add/           添加客户（完整表单）
│   ├── client-detail/        客户详情（信息+跟进记录）
│   ├── house/                房源列表（搜索+状态筛选）
│   ├── services/             交易后服务市场（6大服务+微信弹窗）
│   └── my/                   我的页面（数据统计+菜单）
└── project.config.json       项目配置（AppID已配置）
```

## ⚠️ 合规说明

- 个人小程序无支付功能，服务页面展示微信ID合规
